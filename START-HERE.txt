HOW TO VIEW THE WEBSITE:
================================================================================

METHOD 1: OPEN IN BROWSER (EASIEST)
------------------------------------
1. Open your file explorer/finder
2. Navigate to the "public" folder
3. Double-click "index.html" to open in your browser
4. All links and navigation will work!

OR open "site-map.html" to see links to all pages

METHOD 2: USE LOCAL SERVER (RECOMMENDED FOR DEVELOPMENT)
---------------------------------------------------------
Open terminal/command prompt, then:

For Python users:
  cd public
  python -m http.server 8000
  Then open: http://localhost:8000

For Node.js users:
  npx http-server public -p 8000
  Then open: http://localhost:8000

================================================================================
TROUBLESHOOTING:
================================================================================

If pages don't open:
  - Make sure you're opening files from the "public" folder
  - Try using a local server (Method 2 above)
  - Check that your browser supports HTML5

If images don't load:
  - Images use Unsplash CDN - requires internet connection
  - All image URLs are valid and tested

If styles look wrong:
  - Make sure "styles.css" is in the same folder as HTML files
  - Check browser console for any errors (F12)
